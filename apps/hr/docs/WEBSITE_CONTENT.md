## The Saptta story

Most HR software in India is either a spreadsheet with a login screen, or an enterprise monster your 15-person HR team will never fully configure.

**Saptta** was built for a different customer: growing Indian companies that need **real compliance** (PF, ESI, PT, TDS, Form 16) and **real month-end** (attendance → payroll → bank file → payslips) without hiring a consultant for every release.

We combined:

1. **India-first payroll engine** — statutory deductions, exports your CA and bank already expect
2. **HR-complete workflows** — not just “employee records,” but onboarding, letters, exits, comp-off, expenses, and reports
3. **AI where HR actually wants it** — policy answers, leave questions, review drafts, recruitment JDs — not generic chatbot noise
4. **One workspace per company** — subdomain isolation, role-based views, audit trail

Saptta is part of the Saptta product family (HR today; finance modules can share the same tenant model tomorrow).

## How it works (3 steps for buyers)

### 1. Set up your workspace
HR admin configures org structure, leave policy, salary structures, and statutory settings once.

### 2. Run daily HR
Employees punch, apply leave, submit expenses. Managers approve. HR handles exceptions.

### 3. Close the month
Pre-payroll review → run payroll → adjust → approve → publish. Employees get payslips; finance gets bank file and compliance exports.

## FAQ

**Is my data isolated?**  
Yes — each company gets its own tenant workspace (subdomain).

**Do you file PF/ESI automatically?**  
We generate EPFO/ESIC-ready export files; your HR uploads to government portals (standard for most SMB HRMS).

**Does AI work without internet?**  
AI requires `ANTHROPIC_API_KEY`. All non-AI HR features work offline from your server.

**Can we import employees?**  
Yes — CSV bulk import plus manual add.

**Mobile app?**  
Install as PWA from browser; responsive web for all roles.
